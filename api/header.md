### 接口返回格式

## 单条数据:
> 异常:
```javascript
{ 
    "status":1000,
    "data":{},
    "msg":"错误消息"
}
```
> 正常:
```javascript
{
    "status":0,
    "data":{"id":"1","name":"wb"},
    "msg":""
}
```
## 多条数据:
> 异常:
```javascript
{
    "status":1000,
    "data":{},
    "msg":"错误消息"
}
```
> 正常:
```javascript
{
    "status":0,
    "data":{
        "list":[
            {
                "id":267,
                "city":"Wuhan"
            },
            {
                "id":266,
                "city":"Nanjin"
            }
        ],
        "page":{
            "totalNum":4,
            "totalPage":2,
            "currentPage":1
        }
    }
}
```
